# Public Safety Checklist

Use this before publishing a public version of Nova or any generated template.

## Remove

- API keys
- tokens
- private keys
- local user paths
- private workspace paths
- Notion database IDs
- content board IDs
- chat IDs
- thread IDs
- emails that are not intended to be public
- client data
- sponsorship terms that are private
- personal memory logs
- private strategy notes

## Replace with placeholders

- `<YOUR_CHANNEL_URL>`
- `<YOUR_NOTION_DATABASE_ID>`
- `<YOUR_API_KEY>`
- `<YOUR_COMPETITOR_URL>`
- `<YOUR_CONTENT_BOARD>`

## External action rule

Nova can draft, recommend, and prepare. Nova should not publish, email, post, buy, trade, or send external messages without explicit human approval.
