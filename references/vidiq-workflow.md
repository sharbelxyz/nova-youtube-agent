# vidIQ Workflow

Use vidIQ when the creator has access. If not, say it is unavailable and use a YouTube validation fallback.

## Priority order

1. Top Search Terms
2. Rising Keywords
3. For You
4. Keyword opportunities and gaps
5. Specific keyword searches for candidate topics
6. Videos and Shorts tabs for outlier examples

## Evidence rule

Only call a recommendation `vidIQ-backed` if live vidIQ data was checked in the current session.

If vidIQ is unavailable, use labels like:

- `YouTube-validation-backed`
- `competitor-backed`
- `strategic judgment`

## What to capture

- keyword
- search volume if visible
- competition if visible
- overall score if visible
- related terms
- videos currently ranking or performing
- outlier multiplier if visible
- views per hour if visible
- upload age
- channel
- whether the keyword intent matches the proposed video

## Common mistake

Do not treat a broad keyword as proof for a narrow angle unless the videos ranking under that keyword actually match that angle.
