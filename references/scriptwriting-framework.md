# Scriptwriting Framework

Core rule: the viewer clicks with expectations. The script must confirm the click quickly, then beat the expectation with proof, demos, specificity, and a clearer worldview.

## Before writing

Define target viewer, viewer pain, viewer desire, working title, title expectation, thumbnail expectation, proof available, what would disappoint the viewer, and what will exceed expectations.

## Outline gate

Each major point needs:

- what it is
- why it matters
- how it works in practice
- why it is not generic
- what the viewer sees on screen
- transition into the next point

Do not write the full script if the outline is generic.

## Intro

The intro should include immediate click confirmation, tension, sharper take, proof, and plan.

Keep the delivered script clean. Do not expose these labels in spoken copy.

## Body

Each section needs context, application, framing, and re-hook.

## Outro

End with transformation, not a list recap. Remind the viewer what changed and what to do next.

## SEO package

Every full script should include SEO title, description, tags, chapters, title variants, thumbnail concept, and optional pinned comment.
